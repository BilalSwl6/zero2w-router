# Custom PCB for Raspberry Pi Zero 2W with Case

A custom board that have some button (6) and some leds for status. it also have oled space to connect and see addtional thing on oled.

**Quick Features**

- **Form factor:** Raspberry Pi Zero 2 W-compatible mounting and header alignment.
- **Controls:** 6 tactile buttons for custom actions (programmable actions).
- **Status:** Several on-board LEDs for quick visual feedback.
- **Display:** Dedicated OLED area to mount
- **Cooling:** Heatsink (Alumium) and also enough Space and separate square cut for Fan.

---

The button behaviour is controlled by zerogpio (python), and intended to run as simple services that start automatically on boot.

I also Design CAD Files of Case in Onshape. You can read more about CAD in README file in [`/CAD`](/CAD/README.md) Folder.
