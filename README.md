# Custom Board for Raspi Zero 2W
A custom board that have some button (6) and some leds for status. it also have oled space to connect and see addtional thing on oled.

Actions on button click wil defined via zerogpio in python and will create services that will auto start on raspi start. 

Addtionally i also design CAD Design for raspi and board with fan and OLED space.
